# Sales Performance Dashboard

![Sales Performance Dashboard](Screenshot%202026-04-19%20143252.png)

## 📊 Project Overview
This Sales Performance Dashboard was built in Microsoft Excel for the period **January – August 2026**. It shows key sales metrics, revenue breakdown by category and city, payment methods, and monthly trends.

## ✨ Key Features
- Total Revenue: **₦296,300**
- Total Orders: **15**
- Average Order Value: **₦19,753**
- Interactive charts: Revenue by Category, Revenue by City, Payment Method (Pie), and Monthly Trend
- Slicers to filter by City and Category

## 🔍 Key Insights
- **Electronics** is the top performer generating **₦161,500** (≈54.5% of total revenue)
- **Lagos** has the highest revenue, followed by **Abuja**
- **Card** is the most popular payment method at **43.13%**, followed by Cash (35.44%) and Transfer (21.43%)
- Revenue peaked in **June/July** but dropped in **August**
- Average Order Value is strong at **₦19,753** despite only 15 total orders

## 🛠️ Tools Used
